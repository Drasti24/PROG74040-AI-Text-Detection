# AI-Generated Text Detection

## Overview

This project detects whether a piece of text is human-written or AI-generated using Natural Language Processing (NLP) and Machine Learning.

## Models

### Traditional Machine Learning
- TF-IDF + Logistic Regression
- TF-IDF + Support Vector Machine (SVM)

### Transformer Models
- RoBERTa
- DeBERTa

## Project Structure

```
data/          Dataset (not uploaded to GitHub)
notebooks/     Jupyter notebooks
src/           Python source code
models/        Saved trained models
outputs/       Figures and evaluation results
app/           Deployment application
```

## Dataset

AI vs Human Text Dataset (Kaggle)
